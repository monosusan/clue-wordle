# Clue Wordle

## Inspiration
Clue Wordle was inspired by the popular word game Wordle, with the goal of enhancing the classic experience by adding innovative features that make the game more engaging and challenging. The addition of clues and a timer system aims to create a more dynamic and strategic gameplay experience.

## What it does
- Provides a timed word puzzle experience with smart hint system
- Includes unique clues for each puzzle (text or image-based)
- Implements a timer that turns red after 3 minutes
- Offers color-coded feedback for guesses (green, yellow, gray)

## How I built it
The game is built with several key components:
- A curated dictionary of 600+ words with corresponding clues
- A timer system that tracks solving time
- A toggle-able hint system supporting both text and image clues
- Color-coded feedback mechanism for letter matching
- User interface with clue toggle functionality

## Challenges I ran into
While not explicitly stated in the content, the likely challenges included:

- Balancing game difficulty with hint assistance
- Designing an intuitive user interface that incorporates new features while maintaining familiar Wordle mechanics

## Accomplishments that I'm proud of
